# Web Research

When performing web research tasks:

1. **Search broadly first** — use multiple search queries with different phrasings to get diverse results
2. **Verify across sources** — don't rely on a single source; cross-reference key claims
3. **Structure findings** — organize results with clear headings, bullet points, and a summary
4. **Cite sources** — include URLs for every factual claim so the user can verify
5. **Note recency** — flag when information may be outdated and indicate publication dates
6. **Distinguish fact from opinion** — clearly separate verified facts from editorial content
7. **Summarize at the top** — lead with a TL;DR before diving into details
